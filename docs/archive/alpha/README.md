# Alpha archive — frozen 2026-05-30

This folder is a **point-in-time snapshot of the drop-note alpha phase (S0–S8)**.

Alpha was declared complete on **2026-05-30**: all planned functionality shipped, the full UI redesign landed, and the product moved into **beta** (friend testing for stability before production hardening).

## Rules

- **Frozen.** Nothing here is kept up to date. These files reflect what was true during alpha and are intentionally not maintained.
- **Read-only for history.** Open these only to understand *why* a past decision was made (e.g. why BullMQ was killed in D11, why paid tiers were dropped, why we pivoted positioning). Do not treat any statement here as the current state of the system.
- **Living references live one level up** in `docs/` — `STYLE_GUIDE.md`, `google-oauth-setup.md`, `architecture/email-pipeline.md`. Those are the only docs the project guide (`CLAUDE.md`) points to for current work.
